# Lantern Design System

Ready-to-use React components compatible with our product stack
(React 18.3, TypeScript 5.1, Tailwind CSS 3.4, shadcn/ui).

## Installing a component

Components are distributed via a shadcn registry — the code is copied straight into your project:

```bash
npx shadcn@latest add @lantern/button
```

Add the registry to your project's `components.json` once:

```json
{
  "registries": {
    "@lantern": "https://lantern-registry.vercel.app/r/{name}.json"
  }
}
```
